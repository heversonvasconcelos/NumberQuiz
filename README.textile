h1. NumberQuiz

h2. Descrição

NumberQuiz é um jogo (quiz) simples, onde são apresentadas sequências de números ao jogador e este deve descobrir qual é o termo posterior. 
Aplicação de exemplo que foi implementada com o intuito de fixar algumas questões básicas durante a introdução e aprendizagem da plataforma Java EE. 

h2. Ferramentas utilizadas no desenvolvimento da webapp

* Maven 2
* Tomcat 7
* PostgreSQL 8.4

h2. Frameworks

* JSF2
* PrimeFaces 2
* Spring 3
* Hibernate JPA 1.0
